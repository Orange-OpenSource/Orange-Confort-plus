Confidentialité de vos données personnelles sur Orange Confort+

Cette page a pour objet de vous informer sur la collecte de vos données personnelles par Orange Confort+ et sur les façons dont elles sont utilisées par l’extension.

L'extension Orange Confort+ ne collecte aucune donnée à caractère personnel.

Les données de l’Utilisateur restent dans son Terminal. Pour que l’extension fonctionne correctement, elle doit conserver localement, sur le PC de l’utilisateur, les données suivantes :

  • Le mode d’usage sélectionné par l’utilisateur.
  • Les préférences de réglage de l’utilisateur : réglages de rendus visuels de l’interface (couleurs du texte, taille du texte, couleur de pointeur…), réglages de rendus audios de l’interface (type de vocalisation) et réglages des commandes (ascenseur, boutons de raccourcis…).

Ces données ne sont pas collectées par Orange Innovation Produits & Services.
